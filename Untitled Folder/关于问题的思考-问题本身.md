# 问题的本质

工作和生活中我们会经历各种大大小小的问题，那么问题的本质是什么？

问题的本质：

现实与期望的状况产生了落差。

所谓的解决问题，就是把现实与期望的差距减少、消除。

**问题的类型**

总体而言，问题一般可分为三种类型：

恢复原状型：问题已经发生，与原先的状态产生了差异

防范潜在型：问题还没有发生，但可能会产生问题需要对策预防。

追求理想型：期望改变现状，获得一个更好的状态

明白这些可以帮我们对问题进行划分，排列解决问题的优先顺序。

**问题解决过程中最重要的是什么**

发现问题分析问题

有人会说解决问题执行才最重要，不见得很多问题其他人还没意识到的时候，你发现了，那就是你的机会。

商业大佬，都是趁着整个环境都没有意识到问题的时候，趁机发展壮大自己。比如并多多。



**发现问题和分析问题的方法**

1、问题法

问几个问题

“现状与期待的状态之间有落差？”

“现状有没有发生什么变化？”

“是否觉得哪个部分进行得不顺利”

“是否有些事情未达标准”

“有没有哪些事情不是你原先期待的状态”

“若置之不理，将来是否会发生重大的不良状态”



2、SCQA分析法

描述清楚问题发生的情境

在情境中识别对立冲突的具体事件

分析事件事件的相互关系，确定问题

思考解决答案

3、MECE分析法

对问题拆解定义

讲所有影响要素拆分定义清楚

使这些要素不重复，不层叠，不相交，不遗漏



**面对问题保持平常心**

问题发生时，不要否定现实，延误解决时机

不要上来就追责，先解决问题

面对问题，绝望毫无用处



*办法总比问题多！*
